## About

Member Platform mobile app, written in React Active on Expo.

## Expo.dev

Expo app

## Get Started

To start emulator for testing:

```sh
yarn start
```

## Simulate Clicked Deep Link

Install uri-scheme:

```
npm install -g uri-scheme
```

Simulate link:

```
npx uri-scheme open exp://192.168.0.95:19000/--/two --android
```
